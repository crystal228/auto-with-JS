# Шаги:
* Ввести «Время начала поездки» — 08:00.
* В поле «Откуда»: Усачева, 3.
* В поле «Куда»: Комсомольский проспект, 18.
* Выбрать режим «Свой».
* Выбрать вид транспорта: такси.
* ОР: Текст появившегося результата начинается со слова «Такси»
